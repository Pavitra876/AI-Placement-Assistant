# AI Placement Assistant

An AI-powered web app for MCA students to prepare for campus placements.

## 🚀 Live Demo
https://ai-placement-assistant-one.vercel.app/

## Features
- AI Placement Q&A
- Interview Question Generator
- Resume Analyzer
- Skill Roadmap Generator
- Company-wise Preparation

## Tech Stack
React 19, Vite, Tailwind CSS, OpenRouter AI

## Setup
1. Clone the repo
2. Run: npm install
3. Add .env file with VITE_GEMINI_KEY=your_key
4. Run: npm run dev
